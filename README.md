# Car Dodge Game

A simple 2D retro-style car dodging game built with Flutter.

## Features
- **Endless Gameplay:** Dodge incoming obstacles to survive as long as possible.
- **Dynamic Difficulty:** The game speed gradually increases over time.
- **Scoring System:** Your score goes up based on how long you survive.
- **Cross-Platform:** Works on mobile (touch controls) and web/desktop (keyboard support not explicitly implemented in this simple version, but uses standard Flutter touch/click handling).
- **Responsive Design:** Scales to fit different screen sizes and orientations.

## How to Play
- Tap the **Start Game** button to begin.
- Tap on the **Left** or **Right** side of the game screen to steer your car.
- Avoid the red obstacles. If you hit one, the game is over.
- Try to beat your high score!

## Tech Stack
- Flutter
- Dart
- CustomPainter / AnimatedBuilder for the game loop

## Getting Started

1. Ensure you have Flutter installed on your machine.
2. Clone this repository.
3. Run `flutter pub get` to fetch dependencies.
4. Run the app using `flutter run`.
